# CitySO

Приложение CitySO — это настольный WPF-клиент на .NET 10 для управления квестами, обработки участников и автоматизации ответов через VK.

## Основная функциональность

- Подключение к VK с помощью токена группы.
- Обработка входящих сообщений от участников VK в фоновом режиме.
- Загрузка заданий, участников и ответов из Google Sheets.
- Сохранение и восстановление настроек приложения.
- Управление статусом соревнования и категориями.
- Просмотр состояния подключения по VK и Google Sheets.
- Экспорт / импорт конфигурации в файл `.cityso`.

## Структура приложения

- `CitySO/Program.cs` — запускает хост, регистрирует службы и стартует UI.
- `CitySO/App.cs` — точка входа WPF-приложения.
- `CitySO/Configuration/` — сервис конфигурации, модель настроек и пути к файлам.
- `CitySO/Services/` — бизнес-логика: обработка сообщений, ответы, задачи, пользователи.
- `CitySO/Services/GoogleSheetsServices/` — работа с Google Sheets и авторизация.
- `CitySO/Services/VkServices/` — работа с VK API, отправка сообщений, long poll.
- `CitySO/UI/` — окна и viewmodel для управления приложением.
- `CitySO/BackgroundServices/VkMessagesBackGroundService.cs` — фоновые обработки новых сообщений.

## Требования

- Windows
- .NET 10 SDK
- Доступ в интернет для VK API и Google Sheets API
- Аккаунт VK с группой и ключом доступа
- Google Cloud Console проект со включенным Google Sheets API

## Запуск

1. Откройте решение `CitySO.sln` в Visual Studio или используйте `dotnet`.
2. Из корня репозитория выполните:
   ```powershell
   dotnet build .\CitySO\CitySO.csproj
   dotnet run --project .\CitySO\CitySO.csproj
   ```
3. После запуска откроется главное окно приложения.

## Настройка VK

В приложении откройте **Общие настройки** и заполните:

- `VkApiKey` — токен доступа группы VK.
- `VkGroupId` — идентификатор группы VK.
- `Competition Name` — название соревнования.
- `GoogleSpreadSheetId` — идентификатор Google Sheets, используемого для задач и участников.
- `ShowAnswersImmediately` — если включено, участникам сразу возвращается результат проверки ответа.

> Настройки сохраняются автоматически в `%LOCALAPPDATA%\CitySO\settings.json`.

## Настройка Google Sheets

1. Создайте JSON-ключ учетных данных Google Cloud для OAuth 2.0.
2. В приложении откройте окно **Google Sheets настройки**.
3. Вставьте содержимое JSON-ключа в поле и нажмите **Authorize**.
4. После успешной авторизации приложение сохранит ключ в `%LOCALAPPDATA%\CitySO\googleSheetsKey.json` и токен в `%LOCALAPPDATA%\CitySO\googleSheetsToken.json`.

## Как работает VK-бот

Программа обрабатывает входящие сообщения двумя способами:

- `/help` — выводит справку и доступные команды.
- `/tasks` — показывает список заданий для категории участника.
- `N ответ` — отправка ответа на задание.

Для корректной работы участник должен быть зарегистрирован в Google Sheets и привязан к категории.

### Логика обработки

- `/help` — возвращает информацию о команде и категории участника.
- `/tasks` — возвращает список заданий в категории участника.
- `номер_задания ответ` — проверяет ответ по Google Sheets и сохраняет результат.
- Если соревнование не началось или уже завершено, пользователю будет отправлено соответствующее сообщение.

## Работа с Google Sheets

Приложение использует Google Sheets как источник:

- `TasksService` — загрузка заданий.
- `UsersService` — загрузка участников.
- `AnswersService` — сохранение результатов ответов.
- `CategoriesService` — управление категориями и статистикой.

## Использование интерфейса

### Главное окно

- Просмотр состояния подключения VK и Google.
- Статус соревнования и название.
- Количество обработанных и ожидающих сообщений.
- Кнопки для обновления статуса, импорта/экспорта конфигурации, открытия настроек.

### Общие настройки

- Сохранение названия соревнования.
- VK API токен и ID группы.
- Google SpreadSheet ID.
- Переключатель немедленной выдачи ответа.

### Google Sheets настройки

- Авторизация через JSON-ключ Google.
- Выход для удаления локального токена.

### Управление категориями и статусом

- Добавление новых категорий.
- Изменение статуса соревнования.
- Обновление списка категорий и статистики.

## Импорт / экспорт конфигурации

- Экспортирует файл `.cityso` с текущими настройками.
- Импортирует настройки из файла `.cityso`.

## Логи

- Приложение сохраняет логи в `%LOCALAPPDATA%\CitySO\logs.txt`.
- В главном окне есть кнопка для скачивания логов в произвольный файл.

## Полезные файлы и папки

- `CitySO/CitySO.csproj` — проект приложения.
- `CitySO/Program.cs` — точка загрузки DI и запуска.
- `%LOCALAPPDATA%\CitySO\settings.json` — текущие настройки.
- `%LOCALAPPDATA%\CitySO\googleSheetsKey.json` — Google JSON-ключ.
- `%LOCALAPPDATA%\CitySO\googleSheetsToken.json` — токен Google OAuth.

## Разработка

- Используются пакеты:
  - `CommunityToolkit.Mvvm`
  - `Google.Apis.Sheets.v4`
  - `Microsoft.Extensions.Hosting`
  - `VkNet`
- UI построен на WPF.
- Сервисы зарегистрированы через DI, фоновый воркер обрабатывает новые сообщения.

## Примечания

- Для корректной работы Google Sheets необходимо указать `GoogleSpreadSheetId` и иметь доступ к этому документу.
- VK токен должен иметь права на работу с сообщениями и группу.
- Если приложение не может подключиться, проверьте состояние подключения через кнопки обновления.

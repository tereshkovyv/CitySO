# Сборка CitySO в .exe

Этот документ описывает, как собрать приложение CitySO в самостоятельный .exe файл, который не требует установки .NET Framework.

## Требования

- Windows 10 или выше
- .NET SDK 10.0 или выше (установленный на машине разработчика)

## Способ 1: PowerShell

Откройте PowerShell и выполните:

```powershell
.\build.ps1
```

**Примечание:** Если получите ошибку о политике выполнения скриптов, выполните один раз:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

## Способ 2: Command Prompt (cmd)

Откройте Command Prompt и выполните:

```cmd
build.bat
```

## Опции (только для PowerShell)

Вы можете указать параметры при запуске:

```powershell
.\build.ps1 -Configuration Release -Runtime win-x64 -OutputPath ".\publish"
```

Параметры:
- `-Configuration`: Тип конфигурации (Release, Debug). По умолчанию: Release
- `-Runtime`: Runtime Identifier:
  - `win-x64` - Windows 10 и выше (64-bit)
  - `win-x86` - Windows (32-bit)
  - `win-arm64` - Windows (ARM64)
  - По умолчанию: win-x64
- `-OutputPath`: Папка для вывода. По умолчанию: `.\publish`

## Результат

После выполнения скрипта:

1. Будет создана папка `publish/` (или указанная в `OutputPath`)
2. В этой папке будет файл `CitySO.exe`
3. Этот .exe файл полностью самостоятельный и содержит:
   - .NET Runtime
   - Все зависимости приложения
   - Конфигурационные файлы

## Использование

Просто запустите скомпилированный .exe:

```cmd
.\publish\CitySO.exe
```

Файл можно запустить на любом компьютере с Windows 10 и выше без установки чего-либо дополнительного.

## Размер файла

Самостоятельный .exe файл будет иметь больший размер (~150-200 МБ) из-за включенного .NET Runtime. Это нормально и является ожидаемым поведением.

Если вам нужен меньший размер, можно использовать `PublishTrimmed=true` (удалит неиспользуемый код), но это может привести к проблемам совместимости.

## Дополнительно

- Все файлы конфигурации будут располагаться рядом с .exe в папке `%LOCALAPPDATA%\CitySO`
- Логи записываются в файл `logs.txt` в той же папке настроек
